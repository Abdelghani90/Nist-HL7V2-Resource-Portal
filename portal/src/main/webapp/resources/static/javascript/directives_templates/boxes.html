Welcome to the NIST HL7 V2 Resource Portal


NIST has built and continues to develop a tool kit that supports the testing of HL7 V2 message interfaces based on the concept of message profiles. The foundation of the tool kit is a set of Java APIs that supports testing activities such as automated message generation and message validation. The APIs can be used to build tools, web services, web applications, or they can be incorporated into 3rd party applications and testing environments. Using the APIs, NIST has developed a number of tools.
